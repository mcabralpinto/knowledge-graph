*Means Uniform Resource Identifier. A compact string of characters that identifies a physical or abstract object.*

### Subtypes
- **URL (Uniform Resource Locator):** Specifies the location of a resource on the internet and the protocol used to access it (e.g., `http://example.com`).
- **URN (Uniform Resource Name):** Provides a unique and persistent identifier for a resource without specifying its location (e.g., `urn:isbn:0451450523`).
