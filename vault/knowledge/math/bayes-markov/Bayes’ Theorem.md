*Fundamental concept in probability that describes how to update the likelihood of a hypothesis as more evidence or information becomes available.*

It calculates the conditional probability of event A given that event B has occurred ($P(A | B)$) by combining prior knowledge ($P(A)$) with new data, finding the probability of the evidence given the hypothesis ($P(B | A)$), and normalizing by the overall probability of the evidence ($P(B)$). Thus, its formula is: $$P(A | B) = \frac{P(B | A) \times P(A)}{P(B)}$$This can be generalized to: $$P(A|B, C)) = \frac{P(B|A, C) \times P(A|C)}{P(B|C)}$$where C is additional evidence. In practice, we often use a normalization constant $\alpha$ to simplify calculations: $$P(A | B) = \alpha \times P(B, A)$$
## Naive Bayes
A simplified application of Bayes' Theorem that assumes all features are conditionally independent given the class label. This assumption allows for efficient computation of posterior probabilities, making it suitable for tasks like text classification and spam detection. An important distinction:
- **Marginal Independence:** Two events A and B are marginally independent if the occurrence of one does not affect the probability of the other. Formally, this is expressed as: $$P(A | B) = P(A)$$Or equivalently: $$P(A, B) = P(A) \times P(B)$$
- **Conditional Independence:** Two events A and B are conditionally independent given a third event C if the occurrence of A provides no additional information about B once C is known. Formally, this is expressed as: $$P(A | B, C) = P(A | C)$$Or equivalently: $$P(A, B | C) = P(A | C) \times P(B | C)$$This concept is crucial in simplifying complex probabilistic models, such as Bayesian networks, by reducing the number of dependencies that need to be considered when calculating probabilities.
Essentially: $$P(A | B_1, B_2, ..., B_n) = \frac{P(A) \times \prod_{i=1}^{n} P(B_i | A)}{P(B_1, B_2, ..., B_n)}$$
