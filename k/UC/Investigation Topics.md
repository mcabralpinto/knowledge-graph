## Gen AI - Generative Space Exploration
**High-Level Concept:** A Google Earth-like where images are generated in real time as the user explores an area of the generative space. Aim to solve questions like *How to go beyond the prompt?* and *How to generate truly new things?*.

## Catan
- *Jsettlers:* open-source Java implementation of the Catan rules + hand-coded heuristic-based agent very often used as a base-line in Catan research.
- *Evolvers of Catan (2021):* Improvement on Jsettlers using genetic programming to better inform its decisions. [Microsoft Word - Evolvers of Catan Technical Documentation.docx](https://cs.carleton.edu/cs_comps/2021/game-ai/Final-Results/Evolvers-of-Catan-Website/files/Evolvers-of-Catan-Technical-Documentation.pdf)
- *Playing Catan w/ Cross-dimensional NNs (2020):* Applying a cross-dimensional neural network to create a Catan bot, becoming the first RL agent to achieve better performance than Jsettlers. [2008.07079](https://arxiv.org/pdf/2008.07079)
