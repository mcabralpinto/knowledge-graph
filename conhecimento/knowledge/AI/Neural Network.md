*[[Machine Learning]] model inspired by the human brain. Consists of interconnected layers of operational nodes that simulate neurons, allowing it analyze data and learn patterns.*

Consists of:
- **Input Layer:** Receives raw data and passes it to the next layer.
- **Hidden Layers:** Perform computations and extract features through weighted connections.
- **Output Layer:** Produces the final result, such as classification or prediction.
Each layer has a set number of [[Artificial Neuron]]s.
